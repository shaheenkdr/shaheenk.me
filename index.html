About

My name is Tavis Ormandy, I'm an Information Security Engineer at Google. I'm originally from England, but I'm currently living in Switzerland. My main interests are computing related, I participate in several open source projects and enjoy security research. I sometimes post news or announcements to my weblog, but it probably hasn't been updated recently.

Research

This section is a repository for my UNIX security research and vulnerability analysis, including a list of vulnerabilities I've published. Security is one of my major interests, and I enjoy contributing to open source security by performing audits and research.
